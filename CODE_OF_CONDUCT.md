# Code of Conduct

This is a public policy document and an open repository. The maintainer expects all contributors and commenters to be:

- **Civil.** Disagree without being hostile.
- **Honest.** Disclose conflicts of interest (employment, investment, advocacy positions) in any substantive contribution or critique.
- **On-topic.** This repository is about Finland's sovereign Bitcoin reserve framework. It is not a venue for general crypto debate, political grievance, or unrelated economic theory.
- **Specific.** Vague objections ("this is unrealistic") will be closed without action. Specific objections ("the Track C allocation of 0.5% exceeds Suomen Pankki's stated diversification limits, citing [source]") will be engaged with.

The maintainer reserves the right to close, lock, hide, or delete issues, comments, or pull requests that are abusive, off-topic, spam, or promotional, without explanation.

## Reporting

If you encounter conduct that violates these standards, contact the maintainer directly: jukka@northpoint.fi (or open an issue, if the violation is itself public).

## Scope

This code applies to all repository spaces: issues, pull requests, discussions, and any associated community channels.
